# Razor Pages Bakery Tutorial

This repo contains the code for the application you will build by following the tutorial at [https://www.learnrazorpages.com/razor-pages/tutorial/bakery7](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7). 

This step-by-step tutorial demonstrates how to create a Razor Pages application from scratch, using Visual Studio Code and SQLite, and how to add common functionality to it including communicating with a database, working with cookies, forms and sending email. The steps are as follows:

- Getting started
- [Creating the application](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/new-app)
- [Adding a new page](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/new-page)
- [Working with navigation](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/navigation)
- [Creating the model](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/create-model)
- [Adding a migration](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/migration)
- [Working with data](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/working-with-data)
- [Enabling Sass](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/sass)
- [Starting with forms](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/forms)
- [Working with TypeScript](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/typescript)
- [Using cookies](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/cookies)
- [Creating view components](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/view-components)
- [Adding email](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/email)
- [Using scaffolding](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/scaffolding)
- [Uploading Files](https://www.learnrazorpages.com/razor-pages/tutorial/bakery7/file-upload)
